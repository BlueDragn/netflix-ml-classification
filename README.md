# Netflix ML Classification

## Aim / Purpose
The purpose of this project is to build a baseline machine learning classification model that predicts whether a Netflix title is a Movie or a TV Show based on available model accuracy.

---
## Dataset
The dataset used in this project is a cleaned version of the Netflix Movies and TV Shows dataset.

- The cleaned dataset was produced in **Project 1 (Data Cleaning)**
- It was copied into this repository as a stable input artifact
- Each row represents a Netflix title with associated metadata

---

## Problem Statement
Given metadata about a Netflix title (such as release year, rating, duration, and genres), predict whether the title is a Movie or a TV Show.

This is a binary classification problem.
